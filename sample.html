About StudentProject

A Django multi-app project with containerization and CI/CD.

Project Overview

StudentProject is a Django-based web application designed to demonstrate a multi-app architecture while being fully containerized using Docker. It follows modern software development practices, including CI/CD automation with Jenkins.

Key Features:

  • Multi-app Django architecture (App1 & App2)
  • Dockerized for easy deployment
  • Automated CI/CD pipeline with Jenkins
  • Static web pages with Bootstrap UI
